# Aplikasi Database Rumah Sakit

## Deskripsi
Aplikasi ini merupakan aplikasi database rumah sakit berbasis mobile hybrid (PhoneGap).

## Bahasa Pemrograman
Aplikasi ini menggunakan:
* HTML
* JavaScript (jQuery Mobile)
* PHP
* MySQL
